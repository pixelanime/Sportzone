
# Sports Landing Site

A simple, responsive one‑page website for a sports‑themed brand, aimed at an American audience.  
Built with **HTML5**, **Bootstrap 5**, **Font Awesome**, and a tiny custom CSS/JS file.

## Features
- Hero section with a full‑width background image and call‑to‑action button  
- About section with brief brand story  
- Gallery of popular US sports (basketball, baseball, soccer, football, etc.)  
- Upcoming events carousel (Bootstrap carousel)  
- Contact form (no backend – uses Formspree for quick demo)  
- Fully responsive (mobile‑first)
